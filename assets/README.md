# Mediahouse Website

A complete static website for Mediahouse in a bold black + grunge + purple visual concept. The hero section has been updated to match the approved creator-network dashboard style with a Mediahouse logo, creator cards, campaign stats, purple orbit strokes, and 50+ / 24×7 badges.

## Files
- `index.html` — full website structure
- `styles.css` — complete responsive styling
- `script.js` — menu, reveal animation, email actions
- `assets/` — local SVG graphics and logo

## Brand Color
Primary purple: `#643ABA`

## Free Upload Method: Netlify
1. Go to Netlify.
2. Login / signup.
3. Click **Add new site**.
4. Choose **Deploy manually**.
5. Drag this full folder or the ZIP file.
6. Netlify will give a free live link.

## Free Upload Method: Vercel
1. Create a GitHub repository.
2. Upload all files inside this folder.
3. Import the repo in Vercel.
4. Deploy as a static site.

## Important
This is a static website. The contact form opens the visitor's email app and sends enquiry to `Hello@mediahouse.space`.
To make a real backend form, connect Formspree, Netlify Forms, or a custom backend later.
